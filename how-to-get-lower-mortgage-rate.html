複利計算機按法

使用科學計算機（如 Texas Instruments BA II Plus、HP 12C 或 Casio FC-200V 等專業金融計算機）計算複利的詳細步驟如下。本文以 TI BA II Plus 為例，因其在金融領域應用廣泛：

複利核心公式

複利終值的基本計算公式為：

FV = PV × (1 + i)n

其中：

符號意義說明
FV終值 (Future Value)投資到期時的總金額
PV本金 (Present Value)現在投入的金額
i每期利率如年利率、月利率等
n期數投資的總期數

公式中的 (1 + i)n 稱為複利終值係數，可簡寫為 (F/P, i, n)。實際計算時可透過複利終值係數表直接查詢該值。現值(PV)、利率(i)、期數(n) 可反推公式求出。

casio 計算機複利按法步驟

計算 (1 + i)n 的步驟：

  1. 輸入 1 + i (例如 1 + 0.01 表示利率1%)
  2. 按 × 鍵
  3. 再按 = 共 n-1 次
  4. 若顯示 1.013，就連按兩次 ×=
  5. 乘上本金：按 × PV = 得出終值

範例：現值計算步驟：

  1. 輸入1 + i
  2. 按 / 鍵
  3. 再按 = 共 n 次
  4. 按 × 本金

提示：先測試計算機能否連續 ×=，再執行複利運算。

範例：計算複利終值

  1. [N] 輸入期數，如 3 → N
  2. [I/Y] 輸入利率，如 1 → I/Y
  3. [PV] 輸入本金，如 100000 → PV
  4. [PMT] 輸入 0 → PMT
  5. [CPT] → [FV]

年金情況：若每期有固定支付(PMT ≠ 0)，輸入後可同時計算FV或PV。

注意：臨時現值/終值計算時，注意是否設定為「先付(BGN)」或「後付(END)」模式。

TI BA II Plus 複利按法步驟

步驟 1：重置計算機（避免舊資料干擾）

按 2nd → RESET → ENTER（確認重置）

不同品牌計算機：Casio 按 SHIFT + AC，HP 12C 按 f + CLEAR FIN

步驟 2：設定複利頻率（關鍵！）

按 2nd → P/Y（進入支付/複利設置）：

  • 螢幕顯示 P/Y =（預設值可能是 12）
  • 輸入 12（按月複利）→ 按 ENTER
  • 按 向下箭頭 ↓ 到 C/Y =（複利頻率）
  • 輸入 12 → 按 ENTER

按 2nd → QUIT 退出設定

步驟 3：輸入已知參數

參數按鍵順序螢幕顯示
PV（本金）10000 → +/- → PVPV = -10,000
I/Y（年利率）5 → I/YI/Y = 5
N（總複利期數）10 → 2nd → ×P/Y → NN = 120 (10年×12月)
PMT（定期追加）0 → PMTPMT = 0

步驟 4：計算終值（FV）

按 CPT（Compute）→ FV

螢幕顯示結果：FV = 16,470.09

解釋：10年後帳戶餘額 $16,470.09

金融計算機

使用 TVM 模式：

  1. 先輸入本金 PV (正值或負值按簿記模型)
  2. 設定期數 N 與利率 i (%/期)
  3. 執行 n → i → PV → f + FV (或反向計算 PV)

間隔複利 (如月或季) 設定：

  1. g A 儲存期數
  2. g C 儲存利率
  3. 系統自動計算 n = 年 × 期數，i = 年利率 ÷ 期數

關鍵注意事項

現金流方向

• 投入資金（PV）用 負號（+/-）

• 收回資金（FV）顯示為 正號（表示現金流入）

複利頻率設置（C/Y）

• 年複利：設置 P/Y = C/Y = 1 → N = 10（直接輸年數）

• 季度複利：設置 P/Y = C/Y = 4 → N = 40（10年×4季）

定期追加投資（PMT）

如需每月追加 $500：

輸入 500 → +/- → PMT

結果會變為 $98,347.06（本金+追加+複利）

常見問題解答

Q1：如何計算單利？

複利計算機不支持單利模式，需手動計算：

單利終值 = PV × (1 + r × t)

範例：10,000 × (1 + 0.05 × 10) = 15,000

Q2：結果為何與Excel/網頁計算機不同？

• 檢查是否設置相同複利頻率（C/Y）

• 確認是否輸入了 PMT 或 FV 的初始值

• 建議重置後重新輸入參數

Q3：無金融計算機怎麼辦？

普通科學計算機用公式：

10000 × (1 + 0.05/12)(12×10)

1+0.05÷12=yx120×10000=16470.09
